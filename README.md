# JAVA-Language-Basic-Practice-Codes

So I've been learning Java and decided to save all my practice programs here. Nothing fancy, just the stuff I coded while figuring things out. Thought someone else might find them useful too!

What's in here?
Pretty much everything I covered:

The basics - printing stuff, variables, taking input

Loops & conditions - if-else, for, while (you know, the usual)

OOP stuff - classes, objects, inheritance (took me a while to get this!)

Common programs - patterns, calculators, palindrome check, etc.

File handling - reading/writing files (pretty cool actually)

Why this repo?
Honestly, when I started Java, I wished I had simple working examples to look at. Not the super complex stuff, just clean code that actually runs. That's exactly what I've put here.
